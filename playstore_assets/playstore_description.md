# Deskripsi Play Store - Catatan Keuangan

## Judul App
Catatan Keuangan

## Short Description
Aplikasi catatan keuangan praktis untuk mengelola pemasukan dan pengeluaran dengan mudah.

## Full Description

### Fitur Utama:
✅ **Catat Pemasukan & Pengeluaran** - Catat setiap transaksi dengan mudah dan cepat

✅ **Kategori Lengkap** - Makanan, Transportasi, Kamar (untuk penginapan), Hiburan, Tagihan, Kesehatan, dan lainnya

✅ **Multi Dompet** - Kelola beberapa dompet/wallet sekaligus

✅ **Budget Bulanan** - Tetapkan batas pengeluaran per kategori setiap bulan

✅ **Statistik Grafik** - Lihat pengeluaran dalam bentuk pie chart

✅ **Transaksi Berulang** - Otomatis catat transaksi berulang (kontrak kamar, tagihan bulanan)

✅ **Export PDF** - Ekspor laporan keuangan dalam format PDF profesional (format akuntansi US)

✅ **Backup & Restore** - Backup data JSON untuk keamanan

### Format:
- Format angka: US (contoh: 100,000.00)
- Format tanggal: dd/MM/yyyy
- Negatif dalam tanda kurung: (50,000.00)

### Keunggulan:
🔹 UI Simple dan intuitif
🔹 Data tersimpan lokal di HP
🔹 Tidak butuh internet
🔹 Privacy terjamin - data tidak dikirim ke server manapun
