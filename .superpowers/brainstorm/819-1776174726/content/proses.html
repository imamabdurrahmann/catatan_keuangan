Seberapa Kompleks Prosessnya?

Robot cukuphandle step dasar atau perlu lebih?

A

Basic (3 steps)

Ambil piring → semprot air/sabun → keringkan. Fokus ke automating "masuk" dan "keluar".

B

Standard (4-5 steps)

Ambil → rendam (air + sabun) → gosok → bilas → keringkan. Full cycle seperti dishwasher.

C

Advanced (6+ steps)

Sortir ukuran → rendam → pre-wash → gosok → bilas 2x → sanitize (UV/panas) → kering → sortir ke rak. Semi-industrial.
