3 Pendekatan Desain

Berdasarkan profil kamu: pemula, hobby, flexible budget, arm robot

A

Phased Build (Recommended)

Fase 1: Arm 4-DOF (base rotate, shoulder, elbow, gripper) + dry run tanpa air. Belajar inverse kinematics dasar + servo control.

Fase 2: Tambah water tank + pompa + solenoid valve. Cycle: rendam → gosok → bilas.

Fase 3: Tambah sensor proximity + feedback loop.

Kenapa saya rekomendasikan: Belajar step-by-step, setiap fase bisa work independently, hasilnya bisa di-share bertahap.

B

Minimal Frame Design

Arm SCARA-style (2-DOF horizontal + gripper) dengan linear rail untuk naik-turun. Lebih murah karena pakai motor DC + encoder daripada servo mahal.

Water tank integrated di base. Butuh mekanik lebih banyak tapi kontrol relatif lebih simpel karena SCARA kinematics mudah dihitung.

Kenapa: Lebih murah dan sturdy, tapi range motion lebih terbatas.

C

Full-Featured Robot

Arm 5-DOF dengan stepper motor + optical encoder, suction gripper, water heating, UV sanitization, dan display LCD untuk status.

Full cycle: sort → rendam → gosok → bilas 2x → sanitize → kering → rak. Bisa handle multiple dishes sekaligus.

Kenapa: Paling impressive, tapi kompleksitas tinggi untuk pemula — butuh waktu lebih lama dan banyak debugging.
